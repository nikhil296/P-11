10. Colors : 
                - RGB  : rgb (red, green, blue)
                         rgb(0, 0, 0) is Black, rgb(255, 255, 255) is White.
                - RGBA : rgba (red, green, blue, alpha)
                         alpha - opacity " 0 (fully transparent) to 1 (opaque) ".
                - HEX  : hexadecimal representation.
                         " #rrggbb "  where Black is #000000 and white is #ffffff.
                - HSL  : hsl (hue, saturation, lightness)
                         hue - (0 - 360) where 0 is red, 120 is green, 240 is blue.
                         saturation - it is a percentage value, 0% means a shade of gray,
                                      and 100% is the full color.
                         lightness - it is also a percentage value, 0% is black, and 100% is white.
                
                - HSLA : hsla (hue, saturation, lightness, alpha)
